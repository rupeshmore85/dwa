DWA Project 3 Proposal - Event Calendar

Title = "Event Calendar"

Users can use the Calendar to schedule events in their calendar. Below is the detailed description

  1. Creating a event calendar similar to Google Calendar wherein it displays important events.
  2. After clicking on the relevant date, a description for the event will occur either on the side of the calendar or a popup will appear displaying the results.
  3. The events can be stored in external .js files or the backend database using AJAX.
  4. The user can add events,update events & delete events.
  5. It will display holidays in US and after clicking on it, it can display the detailed holidays.
  6. The calendar displays different skins/formats.
