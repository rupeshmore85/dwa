Project 4 Proposal :- Event Calendar - Integration

Title = "A Complete Event Calendar"

P4 project will be an extension to functionalities used in P2 and P3 projects. User sign up & login functionality will be implemented in order to use the event calendar. The event calendar built for P3 will now be specific to each user who is signed up. He/She can add events to his/her calendar and can integrate those events with Google Calendar. The events can also be shared on the social networking sites.

Features:

  1. Ability to integrate user's event calendar with Google Calendar and Microsoft Outlook in order to sync/export the events from the user's event calendar.
  2. Ability to share the events in the calendar to the users Facebook profile.
  3. Setup new meetings on your event calendar and integrate with popular groups like Meetup group for other users in your group to see your events.
  4. Ability to share your events with other friends who you follow and push the events in their calendar .
  5. Email feature of sending reminders to their email id say on a timely basis prior to the event occurrence.
  6. The Calendar will be available with different predefined skins. To customize the color scheme for the skin, a skin Builder that will allow user to define colors of each element of the calendar interface.
  7. Ability to set recurring events in the calendar.
  8. The Calendar will be made available with 4 different languages (English, French, Spanish & Hindi).
